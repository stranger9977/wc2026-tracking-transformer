Methodology

Full math, modelling choices, and limitations. This recreates Bransen & Van Haaren (MIT Sloan, 2020) using PFF event data from the 2022 FIFA World Cup. References to "the paper" point to that work.

1. From events to action value (VAEP)

Every on-ball event in PFF is converted to SPADL (Soccer Player Action Description Language) form. We then train two gradient-boosted classifiers over a 3-action context window:

  • P-score: probability the possessing team scores within the next 10 actions.
  • P-concede: probability the possessing team concedes within the next 10 actions.

VAEP action value for action \(a\) is the change in expected goal difference it produces:

\[ V(a) = \big( P_{\mathrm{score}}(s_a) - P_{\mathrm{score}}(s_{a-1}) \big) - \big( P_{\mathrm{concede}}(s_a) - P_{\mathrm{concede}}(s_{a-1}) \big) \]
Python implementation (excerpt) 
# research/src/chemistry/vaep/model.py
# For each action, the "prior state" must be flipped if the previous
# action belonged to the opposing team (so prior P_score / P_concede
# swap from the current team's perspective).
same = (teams == prev_team)
prev_value_self_persp = np.where(same, prev_pS - prev_pC, prev_pC - prev_pS)
df["vaep_value"] = (pS - pC) - prev_value_self_persp

2. Joint Offensive Impact (JOI)

For an ordered pair of consecutive actions \((a_i, a_{i+1})\) by teammates \(p\) and \(q\), both VAEP values contribute to JOI(p, q). We sum across all such interactions in the corpus.

\[ \mathrm{JOI}(p, q) = \sum_{(a_i, a_{i+1}) \in S_{p, q}} V(a_i) + V(a_{i+1}) \]
Python implementation (excerpt) 
# research/src/chemistry/joint/joi.py
INTERACTION_TYPES = {"pass", "cross", "dribble", "take_on", "shot"}
for i in range(n - 1):
    if types[i] not in INTERACTION_TYPES or types[i+1] not in INTERACTION_TYPES:
        continue
    if teams[i] != teams[i+1] or players[i] == players[i+1]:
        continue
    lo, hi = sorted((players[i], players[i+1]))
    pair_sum[(lo, hi)] += vaeps[i] + vaeps[i+1]

# Per-90 normalization
joi90 = joi * 90.0 / minutes_together

JOI is normalized by minutes the pair shared on the pitch: \( \mathrm{JOI90}(p,q) = 90 \cdot \mathrm{JOI}(p,q) / m_{p,q} \).

3. Joint Defensive Impact (JDI)

Match event data describes the actions that happened, not the actions a defender prevented. JDI sidesteps this by giving credit to a same-team pair when their opponents underperform their expected offensive impact. For every opposing player \(o\) the pair faced, we compute the gap between \(o\)'s expected and actual OI and divide it across same-team pairs by a responsibility share derived from the 5×5 position grid.

Per-match offensive impact of player \(o\):

\[ \mathrm{OI}_m(o) \;=\; \sum_{a \in \{\text{pass, cross, dribble, take-on, shot}\}} V(a) \]

Expected OI uses player \(o\)'s per-90 mean across prior matches in this dataset, Bayesian-shrunk toward a positional prior (GK / DEF / MID / FWD) when fewer than \(M_0 = 700\) minutes have been played.

Responsibility share. Each on-pitch teammate has individual share \( r_p(o) = (1/d(p, o)) \,/\, \sum_q (1/d(q, o)) \), where \(d\) is Euclidean distance in the 5×5 grid after mirroring the opponent's cell into our coordinate frame (so our defenders land near their forwards). The pair share is the simple mean \( R_m(p, q, o) = \tfrac{1}{2}\big(r_p(o) + r_q(o)\big) \).

Per-match JDI:

\[ \mathrm{JDI}_m(p, q) \;=\; \sum_o \big( \mathbb{E}[\mathrm{OI}_m(o)] - \mathrm{OI}_m(o) \big) \cdot R_m(p, q, o) \cdot \frac{\mathrm{mins}_m(p, q, o)}{90} \]
Python implementation (excerpt) 
# research/src/chemistry/joint/jdi.py
def _mirror_position(pos):
    r, c = grid_cell(pos)               # 0..5 rows, 0..4 cols
    return ((4 - r) if r <= 4 else 5, 4 - c)

def _responsibility_per_opponent(team_players, opponent_pos):
    distances = {p.id: 1.0 / (grid_distance_vs_opponent(p.pos, opponent_pos) + 0.5)
                 for p in team_players}
    total = sum(distances.values())
    indiv = {pid: w / total for pid, w in distances.items()}
    # pair share = (s_p + s_q) / 2
    return {(p, q): (indiv[p] + indiv[q]) / 2 for p, q in pairs(team_players)}

def jdi_m(pair_p, pair_q, opponents):
    return sum(
        (E_OI[o] - OI[o]) * RESP[(pair_p, pair_q, o)] * mins_m[(pair_p, pair_q, o)] / 90
        for o in opponents
    )

Aggregated across matches and normalized per 90 shared minutes gives JDI90.

4. Team builder (mixed-integer program)

Given pair-chemistry scores for every two-player combination in a squad, Bransen's optimal-XI selection is a 0/1 MIP:

\[ \max_{x \in \{0,1\}^{|P|}} \sum_{p \lt q} \mathrm{Chem}(p, q) \cdot x_p x_q \] \[ \text{s.t.} \quad \sum_p x_p = 11, \quad \sum_{p \in \mathrm{GK}} x_p = 1, \quad \sum_{p \in \mathrm{DEF}} x_p = n_{\mathrm{DEF}}, \ldots \]

The product \(x_p x_q\) is linearized via a standard pair-indicator auxiliary variable \(y_{pq} \in [0,1]\) with constraints \(y_{pq} \le x_p,\ y_{pq} \le x_q,\ y_{pq} \ge x_p + x_q - 1\).

Python implementation (excerpt) 
# research/src/chemistry/teambuilder/builder.py
# Formation-enumeration variant of Bransen's MIP.
FORMATIONS = {
    "4-3-3":   {"GK": 1, "DEF": 4, "MID": 3, "FWD": 3},
    "4-4-2":   {"GK": 1, "DEF": 4, "MID": 4, "FWD": 2},
    "3-5-2":   {"GK": 1, "DEF": 3, "MID": 5, "FWD": 2},
    "3-4-3":   {"GK": 1, "DEF": 3, "MID": 4, "FWD": 3},
    "4-2-3-1": {"GK": 1, "DEF": 4, "MID": 5, "FWD": 1},
}

# Pair value combines offense + defense by user-chosen alpha
pair_value[(p, q)] = alpha * joi90[(p, q)] + (1 - alpha) * jdi90[(p, q)]

best = None
for fname, slots in FORMATIONS.items():
    for gk in combinations(by_role["GK"], slots["GK"]):
      for d in combinations(by_role["DEF"], slots["DEF"]):
        for m in combinations(by_role["MID"], slots["MID"]):
          for f in combinations(by_role["FWD"], slots["FWD"]):
              team = list(gk) + list(d) + list(m) + list(f)
              score = sum(pair_value.get((min(a,b), max(a,b)), 0.0)
                          for a, b in combinations(team, 2))
              if best is None or score > best["score"]:
                  best = {"formation": fname, "score": score, "players": team}

5. Limitations

  • Tournament-only baseline. All "expected" values — the Bayesian-shrunk E[OI] used inside JDI, and the per-player Δ OI/90 shown on the Overview — are computed from this 64-match corpus alone. Most players have 2–4 matches here, so the per-player prior collapses to a positional average. That makes our "form" delta a tournament-relative gauge, not a club-vs-national comparison. Closing that loop is the principal next step (extend the loader to StatsBomb / Impect open event data, join on player_id, recompute OI/90, take the difference).
  • Small sample. 64 matches across 32 teams means many pairs share well under 90 minutes. We apply a minimum-minutes filter on the Top Pairs tables to mitigate, but JDI90 in particular is noisy at low sample.
  • Defensive responsibility is a heuristic. Bransen approximates defensive involvement from on-ball recovery / tackle / interception actions in each zone. Off-ball positioning is invisible to event data — a complementary tracking-data line of work lives in src/wc2026_tracking_transformer/.
  • VAEP credit-assignment is action-level. A pass that "creates space" but doesn't directly trigger threat is undervalued. Chemistry inherits this.
  • National-team novelty effect. The very thing we want to study — teams whose players don't play together often — is also the smallest-sample regime. Treat absolute numbers as suggestive, comparisons within a team as more robust.
