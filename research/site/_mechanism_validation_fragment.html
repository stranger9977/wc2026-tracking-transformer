Mechanisms we can see in the data

For each of the 12 named chemistry mechanisms, the Whiteboard already flags one labelled example per featured play. We re-scored those examples against the actual AW-JOI metric — per-frame ball-attention coupling weighted by ΔP(score). If a mechanism is chemistry-positive when it plays out, the affected pair's AW-JOI on the move's window should sit above the same pair's clip baseline.

AW-JOI(p,q,t) = attnball→p(t) · attnball→q(t) · max(ΔPscore(t), 0). Window = ±10 frames around the move's peak; baseline = rest of the clip; lift = in-window / baseline.

Six of the nine open-play mechanisms have at least one labelled example where AW-JOI on the move's window exceeds the rest of the clip — 13 of 15 mechanism-tagged moves clear lift > 1.

Three mechanisms — Meat Wall, Near-Post Flick-On and Short-Corner Overload — cannot be validated yet: none of the five currently featured Whiteboard plays are corners. A corner-clip slot is the obvious next addition. Overlap / Underlap, Press Trap and Positional Rotations have no top-7 candidate move tagged on the existing five plays, so absent rather than negative.

Sample size is tiny — 5 clips, 13 mechanism-tagged moves. This is a structure check, not a population-level effect. The partner in each pair is chosen post-hoc as the strongest same-team coupling inside the window; the meaningful comparison is the same pair's AW-JOI in-window vs out-of-window.
